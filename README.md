# 👶 オンギャーマネジメント

> **アンガーマネジメント**には「怒りを感じたら **6秒** 待つと収まる」という有名な法則があります。  
> 赤ちゃんの泣き声「**オンギャー**」＋「**アンガーマネジメント**」＝ **オンギャーマネジメント**🍼  
> 6秒以内に迷路を解いて、赤ちゃんを泣き止ませよう！

## 🎮 ゲーム概要

- 迷路をクリアして🍼ゴールを目指す、シンプルなタイムアタックゲーム
- 全**7ステージ**構成。ステージが進むほど迷路が大きく、制限時間が短くなる
- 1ファイル（`index.html`）完結で動作。インストール不要

## ⏱ ステージ詳細

| ステージ | 迷路サイズ | 制限時間 |
|---------|-----------|---------|
| 1〜2    | 5 × 5     | 6.0 秒  |
| 3〜4    | 7 × 7     | 5.5 秒  |
| 5〜6    | 9 × 9     | 5.0 秒  |
| 7       | 11 × 11   | 4.5 秒  |

## 🕹 操作方法

| 環境 | 操作 |
|------|------|
| スマホ・タブレット | 画面を指でなぞる |
| PC | 矢印キー / WASD |

## 🚀 遊び方

`index.html` をブラウザで開くだけで遊べます。

```
# リポジトリをクローン
git clone https://github.com/mori-foy/ongya.git

# index.html をブラウザで開く
```

## 🛠 技術スタック

- **HTML / CSS / JavaScript**（ライブラリなし）
- **Web Audio API** — BGM・効果音をすべてコードで生成（外部音源ファイルなし）
- **Canvas API** — 迷路の描画
- Wilson's algorithm による迷路自動生成

## 📣 シェア機能

クリア・ゲームオーバー時に X（Twitter）・LINE へ結果をシェアできます。
